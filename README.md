**Overview**

------

ID3算法例程。

**Obtaining source code**

```shell
git clone git@github.com:xuchunYang1/ID3_Algorithm.git
```

**Build**

需要安装cmake。

```shell
cd ID3_Algorithm
cmake .
make
./ID3
```
